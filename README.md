전북대학교 학생들을 위한 공지사항 및 식단 확인 및 키워드 알림 앱입니다.

### Features

- 학교의 공지사항 및 식단표를 앱에서 바로 확인 가능합니다.
- 특정 글자가 포함된 공지사항이 올라왔을 때 푸시알림을 한다.
- 아침(7:00), 점심(11:00), 저녘(17:00)에 해당 식단표를 푸시알림을 한다.

### 기술 스택

- Lang/Framework: Javascript / react native / serverless
